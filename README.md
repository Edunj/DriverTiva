<h1>SDK for use  a microcontroller board TIVA / Stellaris as a control component microbot</h1>

Author: Eduardo del Nogal Jiménez


It consists of three parts,

* A sdk for using the peripherals involved in controlling microbótico
* A setup PC program for microcontroller peripherals
* A calibration PC program for analog sensors


<h3>features</h3>

* Up to 12 analog sensors
* Up to 16 servo motors, or 4 motors by PWM
* Up to 36 sensors flight time (sonars)
* Up to 2 encoder
* Code for nokia 5110 screen type
* System clock with a 1 us of resolution
